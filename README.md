# 2048-python

## About
Based on the popular game [2048](https://github.com/gabrielecirulli/2048) by Gabriele Cirulli, here is Python version wrapped by [OpenAI gym](https://github.com/openai/gym). 

## Dependencies
- gym
- numpy
- pillow

## Example
```
import puzzle

env = puzzle.Env("your/image/save/dir")
env.reset()
env.render()
```
